## Why this foundation matters

This placeholder post exists so the content pipeline can validate real MDX parsing, frontmatter loading, and syntax highlighting.

### The stack choices

- App Router with Server Components by default
- File-based content in `content/`
- Shared design tokens in CSS custom properties

```ts
export const foundation = {
  shell: "header-footer-layout",
  content: "mdx-plus-json",
  theme: "class-based-dark-mode",
};
```

> Strong foundations make later page work faster, calmer, and easier to reason about.
