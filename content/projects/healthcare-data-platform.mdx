## Overview

At UnitedHealth Group (Optum), I engineered enterprise software solutions leveraging Java, REST APIs, and microservices to improve data accuracy and operational efficiency across healthcare operations.

### Key Contributions

- Built enterprise software solutions that improved data accuracy across complex healthcare systems
- Designed integration flows across cloud systems and API endpoints in collaboration with product managers
- Worked with microservice architectures to ensure scalability and maintainability

## Technical Approach

The platform connected disparate healthcare data systems through a unified API layer, enabling real-time data synchronization and improved operational decision-making.

Delivered measurable improvements in data accuracy and operational efficiency, enabling healthcare operations teams to make faster, more informed decisions.
