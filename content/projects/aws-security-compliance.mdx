## Overview

At Amazon Web Services, I worked within the Security organization to design and deliver secure, scalable services that improved compliance visibility across enterprise accounts.

### Key Contributions

- Designed scalable cloud services using AWS infrastructure (ECS, Lambda, DynamoDB) and Java Spring Boot
- Defined RESTful API integrations ensuring seamless automation between internal platforms
- Presented risk mitigation and architecture improvement strategies to leadership and partner teams
- Built proofs-of-concept demonstrating cloud automation and system hardening

## Technical Approach

The platform leveraged AWS-native services combined with custom Java Spring Boot applications to provide real-time compliance monitoring and automated remediation across enterprise cloud environments.

### Architecture Highlights

| Area | Technology |
| --- | --- |
| Compute | AWS ECS, Lambda |
| Storage | DynamoDB |
| APIs | RESTful integrations |
| Framework | Java Spring Boot |

## Impact

Improved compliance visibility across enterprise accounts and drove adoption of cloud automation practices across multiple business units through well-crafted demonstrations and clear technical communication.
