## Overview

This project centers on a dashboard and campaign workflow for teams running pricing, messaging, and merchandising experiments.

### Scope

- Designed a responsive dashboard for test setup and result review
- Created reusable card, chart, and filter patterns for high-density product surfaces
- Prioritized accessibility and hierarchy so data stays readable under pressure

### Outcome

The resulting interface gives operators a faster path from hypothesis to insight while preserving a calm, high-trust visual tone.
