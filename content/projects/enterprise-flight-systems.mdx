## Overview

As Lead Software Engineer, I defined requirements and proposed architectures aligning enterprise systems with business objectives and customer workflows for one of the world's largest airlines.

### Key Contributions

- Designed scalable backend services using Java Spring Boot, improving system reliability and extensibility
- Led discovery workshops, solution demos, and proof-of-concepts for cross-functional teams
- Presented architecture recommendations to leadership, clarifying complex designs for diverse stakeholders
- Mentored engineers on technical storytelling and stakeholder presentation skills

## Technical Approach

The work focused on bridging the gap between legacy enterprise systems and modern cloud-native architectures, ensuring high availability and performance at scale.

### Architecture Highlights

| Area | Approach |
| --- | --- |
| Backend | Java Spring Boot microservices |
| Infrastructure | Kubernetes orchestration with CI/CD pipelines |
| Integration | RESTful APIs connecting enterprise platforms |
| Methodology | Agile collaboration with business stakeholders |

## Impact

Delivered measurable improvements in system reliability and extensibility while establishing architectural patterns adopted across multiple engineering teams.
